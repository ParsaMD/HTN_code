## Instructions for Running the Code

1. PUT all files into a single folder on your computer.
2. Make sure `3.ipynb`, `NHANES.csv`, and `3.csv` are all in the same folder.
3. Open `3.ipynb` in Jupyter Notebook or Jupyter Lab.
4. Ensure you have the required Python packages: pandas, numpy, etc.
5. Run all cells. The code will automatically find the CSV files.
6. There are a lot of warnings such as '[LightGBM] [Warning] No further splits with positive gain, best gain: -inf', please wait until the end.
